# Fotoğraf Exif Bilgilerini Görüntüleme Aracı

Bu deneysel komut satırı aracında Rexif kütüphanesi yardımıyla fotoğraf dosyalarının Exif bilgilerini okumaya çalışıyorum. Bu deneyi daha sonra Project Lighthouse Social projesinde bir servis library olarak kullanmayı planlıyorum.

## Çalıştırma

```bash
cargo run -- .\Mrs_Smith_and_Mrs_Brown.jpg
```

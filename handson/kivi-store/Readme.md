# Minik Çaplı Bir Key-Value Store

Bir başka rust pratiği. TCP tabanlı, asenkron çalışan, tamamen string key-value çiftleri taşıyan, dockerize edilebilen
bir key-value store.

## Docker

Dockerize işlemleri.

```bash
# Docker build
docker build -t kivi-store .

# Docker Compose ile Çalıştırma
docker-compose up -d
```
